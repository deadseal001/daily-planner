# daily-planner

## Purpose
A website that can provide a work day scheduler which can help users plan their work events on daily basis. The event list can be saved to local storage and refresh page the save events still persist.

## Built With
* HTML
* CSS
* JavaScipt
* JQuery
* Moment.js
* Bootstrap
* Fontawesome
* GoogleFont
